
MAKEUP BOILERPLATE
-----------------------

###Quick start guide:

Install packages
```sh
$ npm install
```

Run Grunt tasks
```sh
$ grunt default
```

Now, you have:
- all *.html *.css files, images, fonts, js at /dist folder
- running webserver at http://0.0.0.0:3003/

Tech stack:
- SASS
- PostCSS (Autoprefixer)
- JADE
- Webpack
- Babel
